# PracticalTech Advisory — Website

Marketing site for [PracticalTech Advisory](https://practical-tech.online), Carolyn Delaney's solo technology advisory practice.

## Structure

Single self-contained landing page:

- `index.html` — landing page (inline CSS, Google Fonts, no build step)

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
npx serve .
# or
python -m http.server 8000
```

## Deploy

The site is static — deploy the folder as-is to Netlify, Vercel, GitHub Pages, or any static host.
